# 開後端 · 逐步指引（連喺邊度撳乜都列晒）

> 目標：開通「雲端同步」+「導師團即時對話 / 個案研討室」。
> 全程約 10–15 分鐘。要你 Google 帳號授權，所以一定要你親手做。
> 🔑 所有密碼/key 只存喺 Apps Script 入面，唔會入 GitHub。

---

## 第 1 步：開一個 Google Sheet（1 分鐘）
1. 去 https://sheets.new （會開一個新試算表）。
2. 改個名做 **IS Training Log**（撳左上角「未命名的試算表」打字）。
3. 睇網址：`https://docs.google.com/spreadsheets/d/`**`呢一段就係 Sheet ID`**`/edit`
   —— 把中間嗰串複製，貼喺記事本暫存。

## 第 2 步：開 Apps Script 專案（2 分鐘）
1. 去 https://script.google.com → 撳左上角 **「+ 新增專案」**。
2. 改名（撳左上「未命名專案」）做 **is-training-backend**。
3. 左邊會見到一個 `Code.gs`：把入面預設嘅幾行**全部刪走**。
4. 打開電腦上 `IS-TRAINING/backend/Code.gs`，**全選複製**，貼入網頁個 `Code.gs`，撳 **💾 儲存**（或 Ctrl/⌘+S）。

## 第 3 步：貼設定檔（1 分鐘）
1. 左邊撳 **⚙️ 專案設定**。
2. 剔選 **「在編輯器中顯示 appsscript.json 資訊清單檔案」**。
3. 返去編輯器（左邊 `<>`），會多咗個 `appsscript.json` → 把 `IS-TRAINING/backend/appsscript.json` 內容貼入去，儲存。

## 第 4 步：填 3 個機密（指令碼屬性）（2 分鐘）
1. 左邊 **⚙️ 專案設定** → 拉到落底 **「指令碼屬性」** → 撳 **「新增指令碼屬性」**，加以下三個：

| 屬性名稱 | 值 |
| --- | --- |
| `SHEET_ID` | 貼第 1 步抄低嗰串 |
| `ACCESS_TOKEN` | 你自己諗一串密碼（例如 16 位英數亂碼，例：`is2026xK9pQ2mfwd`）|
| `CLAUDE_API_KEY` | 去 https://console.anthropic.com → API Keys → 建立一個，貼入嚟（淨係要對話/研討室先需要；驗完 billing）|

2. 撳 **儲存指令碼屬性**。

## 第 5 步：跑 setup() 建表（1 分鐘）
1. 返編輯器，最上面個 function 下拉選 **`setup`** → 撳 **▶ 執行**。
2. 第一次會彈授權 → 撳 **檢閱權限** → 揀你個 Google 帳號 → （見到「Google 未驗證」就撳 **進階 → 前往 is-training-backend（不安全）**，呢個係你自己個 script，安全）→ **允許**。
3. 執行紀錄見到 `✅ setup done` 即成功（個 Sheet 會多咗 `store`、`sessions` 兩個分頁）。

## 第 6 步：部署做 Web App（2 分鐘）
1. 右上角 **「部署」→「新增部署作業」**。
2. 「選取類型」⚙️ → **網頁應用程式**。
3. 設定：說明隨便；**執行身分＝我**；**誰可以存取＝任何人**（靠 ACCESS_TOKEN 保護）。
4. 撳 **部署** → 再授權（同第 5 步）→ 複製個 **網頁應用程式 URL**（`…/exec` 結尾）。

## 第 7 步：接駁 app（1 分鐘）
1. 手機/電腦開 https://initiatesports.github.io/IS-TRAINING/
2. 入 **「數據」** tab → **雲端同步** → 貼 `/exec` 網址 + 你嘅 `ACCESS_TOKEN` → **儲存設定** → **由雲端拉取**。
3. 搞掂！「導師團」同「研討室」即刻可以對話。

## 第 8 步（可選）：Mac 自動上傳資料
喺 `garmin-sync/` 起一個檔案 `backend.conf`（唔會入 repo）：
```
URL=貼你嘅/exec網址
TOKEN=貼你嘅ACCESS_TOKEN
```
之後每朝 LaunchAgent 會自動把計劃/知識庫/Garmin 推上雲；想即刻推：
```
python3 garmin-sync/push_to_backend.py
```

---

### 卡住點算
- 跑 setup 彈「未驗證」→ 進階 → 前往（不安全）→ 允許（係你自己個 script，安全）。
- 對話彈「未設定 CLAUDE_API_KEY」→ 返第 4 步補。
- 「unauthorized」→ app 入面個 ACCESS_TOKEN 同 Script 屬性嗰個唔一致。
